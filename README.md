# PYCE
Source codes for Ice & Life project and others.

It deals mainly with land cover data handling as rasters and shapefiles. 
A specific module $gee$ handles python API for Google Earth Engine.

Two extra modules allows to perform lidar data processing and DEM production ($mnt$), as well as water shed delineation and analysis ($sheds$)

A final module contains scripts ($pyce$_$scripts$) for Ice &Life project papers as well as Lacs Sentinelles production
